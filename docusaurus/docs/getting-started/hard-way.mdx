You can setup your own server for running locally. What you need to learn are:

1. Install developing tools
1. Install programming language
1. Install Wechaty
1. Run your first chatbot

### Requirements

1. A Linux/macOS/Windows computer
1. Internet connection

### 1. Clone wechaty-getting-started Repository

```bash
git clone https://github.com/wechaty/wechaty-getting-started
cd wechaty-getting-started
```

### 2. Install Dependencies

```bash
# npm install
make install
```

### 3. Run the Bot

> **Notice: Wechaty requires Node.js version &gt;= 10**

```sh
# npm start
#   Or use node to run bot directly
#     node examples/starter-bot.js
make bot
```

You are all set!
