随着社会节奏的不断变快，每天，我们都有越来越多的微信消息需要回复，其中不乏很多重复的问题反复找到我。
作为一个程序员，都是不愿意在重复的工作中浪费自己宝贵的时间的。

这里，将给大家分享如何用 Wechaty 和 OpenAI 创建一个属于你的个人助理，
帮助你来处理那些重复的问题，解放你的时间。

## 我们要做什么？

一个微信助理机器人带有以下功能：

- 我@他，并且提问的时候，自动把准备好的材料等都发出来
- 别人问他的时候，可以自动匹配到常见问题并回答
- 当没有匹配到回答的时候，转接给我
  - 消息在群里且我在群里：@我让我回答
  - 私聊或者我不在群里：把我的名片发出来，引导加我好友
