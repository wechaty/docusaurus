title: Deploy in WeChat Official Account
---

WeChat Official Account is a perfect enforcement of the concept of SoLoMo(Social, Local, Mobile) Marketing and has tonnes of features to interact with the audience. In this how to guide, you will learn how to deploy the bot in WeChat Official Account locally and integrate bot from the list of examples present in [starter templete](https://github.com/wechaty/wechaty-getting-started).

![Deploy in Lark](../../static/img/docs/howto/wechat-offical/wechaty-official-account.webp)

## Requirements

1. Your system must have [Node.js](https://nodejs.org/en/download/package-manager/) installed (version >= 12).
2. Your system must have [Wechaty](https://github.com/wechaty/wechaty) (version >= 0.40).
3. You must be familiar with [Wechaty puppet official account](https://www.npmjs.com/package/wechaty-puppet-official-account).

## Integration

Let's take up an example on how to integrate bot from [starter templete](https://github.com/wechaty/wechaty-getting-started) to WeChat Official Account. The step is similar for all other bots as well.

### Prerequisite

1. Official Wechaty package: [package/wechaty](https://www.npmjs.com/package/wechaty).
2. Address of the service provider that supports HTTP and HTTPS such as 80, 443. Several internet penetration tools: [localtunnel](https://localtunnel.github.io/www/).
3. Permanent external network address: [ngrok](https://www.npmjs.com/package/ngrok).

You can follow up the steps mentioned below:

1. Initialize the project by creating a new folder `my-bot`.

```ts
mkdir my-bot
cd my-bot
```

2. Install the dependencies using the following commands:

```ts
npm install wechaty
```

3. Develop temporary internet address as follows:

```ts
npm install -g localtunnel lt â€“port 80
```

4. Add the dependencies for using the bot with WeChat Official Account:

```ts
npm install wechaty-puppet-official-account
```

5. Create a new folder `src` and add a file `my-bot.js`. Add any of the functions from <a href="#"> add functionality to the bot</a> section to the snippet below:

```ts
import {
  Contact,
  Message,
  ScanStatus,
  WechatyBuilder,
  log,
} from 'wechaty'
console.log(welcome)
const bot = WechatyBuilder.build()
/*
 *Your function goes here
 */
```

6. Configure your `appId`, `appSecret`, `token`, `port` as follows:

```ts
const bot = WechatyBuilder.build({
    name: '<your bot name>',
  puppet: new PuppetOA({
    appId: "wxbd801c28fbe1bbbd",
    appSecret: "6959408a3ba1c82db1a11d941df65764",
    token: "token",
    port: 80
  })
})
```

7. After you are done with the file, you can run the bot using the following commands:

```ts
export WECHATY_LOG=verbose
export WECHATY_PUPPET=wechaty-puppet-official-account
node src/my-bot.js
```
Now, you are ready to play with the bot!
