This document outlines when various pieces of Wechaty will be removed or altered
in a backward incompatible way, following their deprecation.
More details about each item can often be found in the release notes of two versions prior.

## Deprecation Timeline

We clean the code after the end of the year, so it is not possible to see the exact date of deprecation.
