## Classes

### Wechaty

Wechaty 的构造类用于构造 Wechaty 实例。

一个Wechaty Bot 代表着一个微信的客户端，他取决于你具体使用哪一个Puppet。

* **web-WeChat** - 当使用Puppet-Puppeteer 或 Puppet-wechat4u  
* **iPad-WeChat** - 当使用Puppet-padchat
* **iOS-WeChat** - 当使用Puppet-ioscat
